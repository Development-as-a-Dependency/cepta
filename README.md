## What is CEPTA?

CEPTA is a CLI tool that helps you create a new Express.js app with Pug and TailwindCSS.

## Installation

```bash
npm install -g cepta
```

## Usage

```bash
cepta create <project-name>
```

## Example
  
```bash
cepta create my-app
```
